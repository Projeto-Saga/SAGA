# SAGA — Sistema de Gestão Acadêmica

Descrição
---------
SAGA é um sistema web desenvolvido para facilitar a gestão acadêmica em instituições de ensino, permitindo rematrícula, consulta de histórico, visualização de faltas e notas, e envio de documentos para a secretaria.
